TITLE:- Pong Game using raylib
-----------------------------------------------------------------------------------------
A classic 2D Pong implementation built using C++ and the Raylib library. This project  support local two-player gameplay & gameplay with Bot.
-----------------------------------------------------------------------------------------
📁 Project Structure

-code  : All C++ source files (.cpp, .h)
-raylib: Raylib library headers and binaries
-assets: Game resources (graphics, sounds)
-docs  : Technical documentation & notes, UML diagrams, and VS2022 setup guides
-----------------------------------------------------------------------------------------
🎮 Controls

	 	| Move Up  | Move Down 
---------------------------------------
Player 1 (Left) |     W    |     S
Player 2 (Right)| Up Arrow | Down Arrow

Press 'C' to switch between mode 2V2 or 1VBot
-----------------------------------------------------------------------------------------
🚀 How to Run

1. Open the game code folder.
2. Double click on gamecode.sln
3. Build and run the code
-----------------------------------------------------------------------------------------
⚠️ Technical Configuration Notice

If the compiler displays errors regarding "Raylib" (such as red underlines or "file not found" alerts) or if the library is not properly linked upon opening the solution, please refer to the detailed setup guide located at:

docs->config for visual studio 2022.txt

This document provides a step-by-step walkthrough for configuring the Include Directories, Library Directories, and Linker Dependencies required to build this project within the Visual Studio 2022 environment.
